About


Proton Mod Manager is a website (and soon-to-be in-game manager) meant to host Balatro mods online for people to share. It's a rather basic site, as it's more meant as an efficient way to store mod data for a future in-game manager, but generally, it should hold up very well.


How to Access Mod Data

The way to access the raw data for your uses is just by accessing the /data subdomain. There, all the relevant data is stored in a .json file, such as id, name, author(s), description, dependencies, etc. Everything is stored except the latest version or the list of releases/release info, which would quickly get outdated and is better to fetch directly from GitHub when needed.


Favourites

What are "favourites"? Currently, they are unimplemented, though when we implement the in-game version, we will have an implemented like system on both platforms, with each person being able to like whatever projects they wish, and rise to the top of the ranking. It may be implemented sooner, depending on how long it takes to implement the in-game version, so please wait until then.
