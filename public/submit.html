Upload a Project


Submitting a project to the database is rather simple. Simply give the link to your repository below, and the path to the metadata file, and it will be added to the database.

There is one thing to keep in mind: In your metadata JSON file, you must have an "allow_redistribution" variable set to true. This is to prevent other people from uploading creators' mods without their consent.
